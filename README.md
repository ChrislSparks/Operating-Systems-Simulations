# Operating-Systems-Simulations

## Description

A collection of Java projects that simulate fundamental operating system concepts, including process hierarchies, deadlock detection, and file allocation. These projects demonstrate recursive algorithms, resource management, linked structures, and operating system data structures.

## Projects

### Process Hierarchy Simulator
Simulates parent-child process relationships using a process table.

### Deadlock Detection Simulator
Detects deadlocks using recursive resource allocation graph traversal.

### File Allocation Table Simulator
Models linked file allocation and block management used in FAT file systems.

## Technologies

- Java
- Object-Oriented Programming
- Recursive Algorithms
- Operating Systems
